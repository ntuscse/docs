# Environment Setup

Frontend devs should have the following installed and configured as described below:

## NVM (Node version manager)

:::danger
Do not install node.js outside of nvm. if you have node.js installed,
you should uninstall it before proceeding with the nvm install
:::

Testing only

## Yarn v1

Install yarn from [here](https://classic.yarnpkg.com/en/docs/install)

## React config

Add `export REACT_EDITOR=webstorm` to your zshrc/bash_profile to open
webstorm via react error messages
